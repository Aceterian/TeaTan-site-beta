# Tea Ceremony Website

This project was created in collaboration with a team organizing tea ceremonies. My role was to redesign and modernize an existing website while preserving the functionality that was already working reliably.

## My Contributions

* Redesigned the website's UI to match the team's vision.
* Improved the overall layout, structure, and user experience.
* Migrated and integrated the existing Formspree contact/booking form.
* Configured the domain and deployed the website to hosting.
* Performed testing and fixed issues to ensure stable functionality.

## Purpose

Besides helping the team, this project served as an opportunity to refresh my HTML and CSS skills after taking a break from web development. It allowed me to revisit modern web practices while delivering a functional website for a real client.
